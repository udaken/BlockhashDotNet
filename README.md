# BlockhashDotNet
Image perceptual hash calculation in C# http://blockhash.io/

## `BlockhashDotNet.Core` 
A project that just implemented the algorithm.

## `BlockhashDotNet.Winforms` 
Facade for `System.Drawing.Bitmap`.

## `BlockhashDotNet.Wpf` 
Facade for `System.Imaging.BitmapSource`.
